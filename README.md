# Chicago Public School Budget Calculator
## Fiscal Year 2027

#### On May 15, Chicago Public Schools provided principals with the district-funded portion of their FY27 budget. Using this data CTU calculated the positions that will be lost with this years budget allocation formula. We compare this loss to the hypothetical gain in positions if Illinois' Evidence-Based Funding Formular were 100% funded. 

#### The data used to create this app with downloaded, cleanded, and assembled by CTU's Research Department. The streamlit webapp was created by Chris Poulos (Public Finance Policy Analyst, CTU, christopherpoulos@ctulocal1.org).
